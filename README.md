# Claude Docs

Claude Code configuration files and reference docs for development projects.

## Usage
- Copy `CLAUDE.md` into the root of any project and customize it
- Reference `/docs/` files on demand in Claude Code with `@docs/filename.md`
- Copy `/tasks/` folder into each project for task tracking

## Files
| File | Purpose |
|------|---------|
| `CLAUDE.md` | Master Claude Code configuration |
| `docs/plan-mode-prompt.md` | Garry Tan's code review framework |
| `docs/db-connection.md` | PgBouncer + Prisma dual-URL setup |
| `docs/db-rollback-process.md` | Migration rollback procedures |
| `docs/docker-workflow.md` | Docker compose workflow reference |
| `tasks/todo.md` | Active task tracking template |
| `tasks/lessons.md` | Claude self-improvement log |
